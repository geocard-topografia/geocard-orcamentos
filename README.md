# App GeoCard de Orçamentos

## Como publicar no Vercel
1. Crie uma conta no Vercel.
2. Clique em New Project.
3. Envie esta pasta ZIP descompactada ou conecte ao GitHub.
4. O Vercel detecta Vite automaticamente.
5. Clique em Deploy.

## Rodar no computador
```bash
npm install
npm run dev
```

## Gerar versão final
```bash
npm run build
```
